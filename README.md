📊 ABOUT THIS DASHBOARD
A professional-grade Binance Futures algorithmic trading dashboard built in Python with Streamlit. Combines real-time market data, multi-indicator signal generation, ATR-based risk management, and a full paper trading simulator — all in one interface.

🧠 SIGNAL ENGINE
✓ EMA 9 / 21 / 45 / 200 stack alignment
✓ MACD fresh crossover detection
✓ RSI momentum zone filter
✓ Bollinger Band squeeze / breakout
✓ Volume surge confirmation
✓ EMA200 macro trend gate
→ Requires 7 / 10 score to trigger

🛡️ RISK MANAGEMENT
✓ ATR-based dynamic stop loss
✓ Trailing stop — locks in profits
✓ Configurable R:R ratio (1:1 to 1:5)
✓ Position size = % risk / SL distance
✓ 20% capital cap per trade
✓ Post-loss cooldown (no revenge trading)
→ Works on Binance Futures (Perpetual)
